// CAGR = (ending / beginning)^(1/years) - 1
function computeCAGR(beginning, ending, years) {
  return Math.pow(ending / beginning, 1 / years) - 1;
}

Financial Formulas

Present Value (single cash flow):

$$PV = \dfrac{CF}{(1+r)^t}$$

Net Present Value (NPV):

$$NPV = \sum_{t=0}^{T} \dfrac{CF_t}{(1+r)^t} - InitialInvestment$$

Walk through inputs: discount rate 'r' (cost of capital), cash flows, and horizon 'T'.
